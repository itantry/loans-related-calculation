# Home Loan Calculator -- version 3.1

This project is a web-based home loan calculator that allows users to estimate their monthly EMI, compare different loan scenarios, and download the comparison table as a PDF.

## Features

-   **Loan Calculation**: Calculates monthly EMI, estimated interest, actual interest paid (with additional payments), and adjusted tenure.
-   **Comparison Table**: Allows users to add and compare multiple loan scenarios.
-   **PDF Download**: Enables users to download the comparison table as a PDF file.
-   **Interactive Charts**: Visualizes loan data using pie and line charts.
-   **Local Storage**: Remembers user inputs using local storage for a better user experience.
-   **Responsive Design**: Provides a responsive layout that works on different screen sizes.

## Technologies Used

-   HTML
-   CSS
-   JavaScript
-   Chart.js
-   jsPDF
-   html2canvas
